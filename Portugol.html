Portugol

Portugol é uma linguagem de programação simplificada para o entendimento mais fácil, destinada a facilitar o aprendizado de lógica de programação e algorítmos, especialmente para iniciantes. Portugol foi a primeira linguagem de programação que eu aprendi, e continuo aperfeiçoando as minhas técnicas e sempre aprendendo mais sobre.
